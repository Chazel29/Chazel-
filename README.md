# Chazel — Official Website

A 4-page site for Chazel (Iradukunda Chalom), Burundian memelord & content
creator. Built from the same structure as the Kirikou Akili template, reskinned
with a bold/meme visual theme.

## Pages
- `index.html` — Home (hero, featured video, social stats)
- `about.html` — About (bio, quick facts, content stats)
- `partnership.html` — Superbat brand ambassadorship + Adda Global Company
  (co-founded with Bahe and Adelard: travel agency, clearing & forwarding,
  transport & logistics, marketing agency, construction)
- `contact.html` — Contact info + message form

## Files you need to add
Create an `images/` folder next to these files and add:
- `chazel-hero.jpg` — home hero background
- `chazel-about.jpg` — about page hero background
- `chazel-superbat.jpg` — partnership page hero background
- `thumbnail.jpeg` — video poster image
- `facebook.png`, `instagram.png`, `tiktok.png` — social icons
- `gmail.jpg`, `whatsapp.png` — contact icons

And a `videos/` folder with:
- `chazel-latest.mp4` — featured clip on the home page

## Editing
- Colors, fonts, and the "sticker" badge style all live in `style.css` under
  `:root` — change the hex values there to retheme the whole site at once.
- `script.js` is unchanged from the template: it handles the hamburger menu,
  video play/pause, and the contact form's success message.
- Update the phone number / WhatsApp link and email address in `contact.html`
  and `about.html` once you have the final ones you want published.
